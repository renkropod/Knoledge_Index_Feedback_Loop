## 4-1. 신규 개념 3 개
1. **명세 기반 코드 생성 (Spec-based Code Generation)**
   - 정의: 자연어 또는 형식적 명세를 입력받아 코드를 자동으로 생성하는 기술.
   - 왜 지금: AI 에이전트 코딩의 핵심 기술로 부상했으나, '명세만으로는 부족하다'는 반론이 제기됨.
   - 관련 글: A sufficiently detailed spec is code
   - 지식카드 연결: spec-code-gap

2. **에이전트 코딩 옹호론 (Agentic Coding Advocacy)**
   - 정의: AI 에이전트가 명세만으로도 코드를 생성할 수 있다고 주장하는 관점.
   - 왜 지금: AI 코딩 도구의 발전으로 인해 이러한 주장이 확산됨.
   - 관련 글: A sufficiently detailed spec is code
   - 지식카드 연결: agentic-coding-myth

3. **컨텍스트 효율성 (Context Efficiency)**
   - 정의: AI 에이전트가 작업을 수행할 때 소모하는 컨텍스트 (메모리, 토큰) 의 양을 최소화하는 능력.

- **MCP (Model Context Protocol)**: AI 모델과 외부 도구/데이터를 연결하는 프로토콜.

- **컨텍스트 효율성 (Context Efficiency)**: AI 에이전트가 작업을 수행할 때 소모하는 컨텍스트의 양을 최소화하는 능력.
